# ibm-mq-connector

[![CI Build](https://github.com/axonivy-market/ibm-mq-connector/actions/workflows/ci.yml/badge.svg)](https://github.com/axonivy-market/ibm-mq-connector/actions/workflows/ci.yml)

This connector gives you access to one or more IBM MQ message handling servers. It allows you to define multiple, inheriting connection configurations and lets you quickly create a `Consumer` or a `Producer`.

Additionally, this connector provides an IProcessStartEventBean which can be used to start Ivy processes which react on IBM MQ messages synchronously or asynchronously and supported schema registries.

Read our [documentation](ibm-mq-connector-product/README.md).
